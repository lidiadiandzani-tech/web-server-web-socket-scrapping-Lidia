Advanced Web Server + WebSocket + Web Scraping

📌 Deskripsi

Project ini adalah web server berbasis socket Python yang dilengkapi WebSocket dan Web Scraping untuk menampilkan data secara real-time.

🚀 Fitur

* Web Server manual
* WebSocket real-time
* Web Scraping
* UI Bootstrap

▶️ Cara Menjalankan

1. Install:
   pip install -r requirements.txt
2. Jalankan:
   python server.py
3. Buka:
   http://localhost:8081
